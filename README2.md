# Date Difference Calculator

A Python script that calculates the number of days between two dates.

## Description
This project takes two dates as input and calculates the absolute difference in days between them.

## Features
- Date parsing
- Difference calculation
- Beginner-friendly logic

## Technologies Used
- Python
- datetime module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python date_difference.py

4. Enter dates in YYYY-MM-DD format

## Example

Input:
2026-01-01
2026-01-10

Output:
Difference: 9 days
